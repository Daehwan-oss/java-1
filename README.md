## 증감 연산자
    a++ : a를 1 증가 하고 증가 전의 값 반환
    ++a : a를 1 증가 하고 증가된 값 반환
    a-- : a를 1 감소 하고 감소 전의 값 반환 
    --a : a를 1 감소 하고 감소된 값 반환

## 대입 연산자
    a = b : b의 값을 a에 대입
    a += b a = a + b 와 동일
    a -= b a = a - b 와 동일
    a *= b a = a * b 와 동일
    a %= b a = a % b 와 동일
    a &= b a = a & b 와 동일
    a ^= b a = a ^ b 와 동일
    a \= b a = a \ b 와 동일
    a <<= b a = a << b 와 동일
    a >>= b a = a >> b 와 동일
    a >>>= b a = a >>> b 와 동일


비교 연산, 논리 연산
비교연산자 : 두개의 값을 비교하여 true / false 결과

    a < b  : a가 b보다 작으면 true
    a > b  : a가 b보다 크면 true
    a <= b : a가 b보다 작거나 같으면 true
    a >= b : a가 b보다 크거나 같으면 true
    a == b : a가 b와 같으면 true
    a != b : a가 b와 같지 않으면 true

논리 연산자 : 두개의 논리 값에 논리 연산, 논리 결과

    ! a : a가 true 이면 false, false 이면 true  
    a^b : a와 b의 XOR 연산. a,b가 같으면 false
    a \\ b : a와 b의 OR 연산. a 와 b 모두 false인 경우만 false
    a && b : a와 b의 AND 연산. a 와 b 모두 true인 경우만 true

조건 연산
3개의 피연산자로 구성된 삼항(ternary) 연산자
opr1>opr2:opr3 -> opr1이 결과가, true면 opr2, false면 opr3가 나옴

조건문 - 단순 if 문, if-else문

Switch문
Switch문의 식과 case 문의 값과 비교
    ->case의 비교값과 일치하면 해당 case의 실행문장 수행
    ->break를 만나면 switch문을 벗어남
    ->case의 비교 값과 일치하는 것이 없으면 default 문 실행
    ->defalut문은 생략 가능

Case 문의 값
    -> 문자, 정수, 문자열 리터럴(JDK1.7부터) 만 허용
    ->실수, 리터럴은 허용되지 않음.
